# backend-SpringBoot

##  Eclipse에서 확인해야 할 주요 설정들
# 1. Installed JREs 확인 및 설정
Window > Preferences > Java > Installed JREs
jdk-17

# 2. Compiler 설정
Window > Preferences > Java > Compiler

Compiler compliance level을 JDK 버전에 맞게 설정
 = 17 버전
